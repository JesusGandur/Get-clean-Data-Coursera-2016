GETTING AND CLEANING DATA COURSE PROJECT 03/2016

This file describes the run_analysis.R script works, where is developed the course project for
getting and cleaning data from COURSERA.

1. Unzip the data from https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip
set the folder as the working directory
2. use source("run_analysis.R") command in RStudio.
3. you will find at the end of the script the next file generated:

 * data_step5.txt (219 Kb): it contains a data frame called result with 180*68 dimension.
